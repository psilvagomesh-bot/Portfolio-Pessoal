Atualização do Projeto: Portfólio

Nome: Portfólio Pessoal.

Objetivo do portfólio: Apresentar as aplicações em HTML, CSS, JavaScript, projetos desenvolvidos e a responsividade com MediaQueries.

Descrição das Páginas:

Index: Página principal do projeto, onde podemos encontrar uma foto de perfil com uma breve animação e um pequeno texto que
descreve o objetivo pessoal do aprendiz.

Projetos: Seção que contém os projetos desenvolvidos em sala de aula divididos em cards clicáveis.

Contato: Onde será encontrado o formulário de contato com as noções básicas: validação de dados e o envio de mensagem altomática através do EmailJS, junto com ela, a seção de redes sociais com GitHub e Instagram.


Tecnologias Usadas: HTML, CSS, JavaScript e EmailJS.

Estrutura Básica:

HTML:

- Apresenta os títulos, textos, parágrafos de cada página;

- Seções de conteúdos como as Divs, Sections e Article;

- Menu de navegação de Links;

- Implementação de imagens;

- Implementação de Formulário.

CSS:

- Apresenta a estilização geral do projeto com cores, espaçamentos e estilização de textos, imagens, seções de conteúdo e links.

JavaScript:

- Apresenta a interação e movimento do projeto com o usuário, por exemplo, a integração do EmailJs com formulário da página de contato que irá enviar a mensagem automáticamente;

- Apresentação de Popups em cada card da página de projetos após clicar em cada um deles.

Como executar o projeto:

Caso queira executar o projeto, basta acessar o link do mesmo que estará diponível na ativiade proposta no google classroom ou clonar o repositório que está no GitHub e executá-lo no seu PC local.

Melhorias Implementadas:

Atualização na página de projetos, os projetos descrevidos nos cards agora são todos direcionáveis aos seu devidos conteúdos apresentados após clicar em cada um deles.
